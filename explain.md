# Week 5 – JavaScript Fundamentals Explanation

เอกสารนี้จัดทำขึ้นเพื่ออธิบายแนวคิดการทำงานของโค้ดในแต่ละไฟล์ พร้อมยกตัวอย่างผลลัพธ์ที่ได้จากการรันจริง เพื่อให้เข้าใจว่าโค้ดทำงานอย่างไรและได้ผลลัพธ์นั้นมาได้อย่างไร โดยสามารถเปิดอ่านบน GitHub ได้โดยตรง

---

## 2.1 ไฟล์ [01-variables.js](01-variables.js)

### 6. Challenge: Create a Person Object

**ตัวอย่างผลลัพธ์**

```text
Student object: { ... }
Full name: Alice Smith
Info: Alice Smith, Age: 20, GPA: 3.8
Courses: HTML, CSS, JavaScript
```

**อธิบายการทำงานของโค้ด**

* โค้ดสร้าง object ขึ้นมาเพื่อแทนข้อมูลของบุคคลหนึ่งคน
* ภายใน object มีทั้งข้อมูลส่วนตัวและฟังก์ชันสำหรับเรียกใช้ข้อมูล
* ใช้ `this` เพื่อเข้าถึงข้อมูลภายใน object เดียวกัน
* คำสั่ง `join()` ใช้รวมข้อมูลใน array ให้เป็นข้อความบรรทัดเดียว

## 2.2 ไฟล์ [02-functions.js](02-functions.js)

### 8. Returning Objects

**ตัวอย่างผลลัพธ์**

```text
Email: john.doe@example.com
Full name: John Doe
```

**อธิบายการทำงานของโค้ด**

* ฟังก์ชันถูกออกแบบให้สร้าง object และส่งกลับออกมาเป็นผลลัพธ์
* ใช้รูปแบบ shorthand property เพื่อเขียนโค้ดให้สั้นลง
* method ภายใน object ใช้ `this` ในการอ้างถึงข้อมูลของตัวเอง

### 9. Function as Parameter (Callback)

**ตัวอย่างผลลัพธ์**

```text
Original: [1, 2, 3, 4, 5]
Doubled: [2, 4, 6, 8, 10]
Squared: [1, 4, 9, 16, 25]
```

**อธิบายการทำงานของโค้ด**

* มีการส่งฟังก์ชันเข้าไปเป็นพารามิเตอร์ของอีกฟังก์ชันหนึ่ง
* ฟังก์ชันหลักจะเรียก callback กับข้อมูลแต่ละตัวใน array
* ช่วยให้สามารถเปลี่ยนพฤติกรรมการทำงานได้โดยไม่ต้องแก้โค้ดหลัก

## 2.3 ไฟล์ [03-control-flow.js](03-control-flow.js)

### 5. Short-Circuit Evaluation

**ตัวอย่างผลลัพธ์**

```text
User name: John
User profile: undefined
```

**อธิบายการทำงานของโค้ด**

* ใช้ optional chaining (`?.`) เพื่อป้องกัน error เมื่อค่าที่เรียกเป็น null
* ตัวดำเนินการ `||` จะเลือกค่าที่เป็นจริงตัวแรก
* เมื่อเจอค่าที่ใช้งานได้แล้ว JavaScript จะหยุดตรวจทันที

### 7. Form Validation

**ตัวอย่างผลลัพธ์**

```text
Valid user: { isValid: true, errors: [] }
Invalid user: { isValid: false, errors: [...] }
```

**อธิบายการทำงานของโค้ด**

* ตรวจสอบข้อมูลแต่ละช่องด้วย if/else
* หากข้อมูลไม่ถูกต้องจะเก็บข้อความแจ้งเตือนไว้ใน array
* สุดท้ายดูจำนวน error เพื่อสรุปว่าฟอร์มผ่านหรือไม่

## 2.4 ไฟล์ [04-loops.js](04-loops.js)

### 9. Chaining Methods

**ตัวอย่างผลลัพธ์**

```text
Even numbers squared: 4, 16, 36, 64, 100
```

**อธิบายการทำงานของโค้ด**

* ใช้ `filter()` คัดเฉพาะค่าที่ต้องการ
* ต่อด้วย `map()` เพื่อแปลงค่า
* และ `join()` เพื่อรวมผลลัพธ์เป็นข้อความเดียว

### 10. Challenge: Student Grades

**ตัวอย่างผลลัพธ์**

```text
Names: Alice, Bob, Charlie, Diana, Eve
Class average: 87.00
Top scorer: Alice (95)
```

**อธิบายการทำงานของโค้ด**

* ใช้ `map()` เพื่อดึงชื่อออกมา
* ใช้ `reduce()` ในการคำนวณค่าเฉลี่ย
* ใช้ `sort()` เพื่อเรียงคะแนนจากมากไปน้อย
* ทำให้ทราบนักเรียนที่ได้คะแนนสูงสุด
